# Unsupervised Categorical Embeddings
This repository contains example code for the paper "Unsupervised Embeddings for Categorical Variables" [1].

Requirements:
- Python 3.6
- https://github.com/hannesdm/glove-python
- gensim
- scikit-learn
- pandas


[1] H. De Meulemeester and B. De Moor, "Unsupervised Embeddings for Categorical Variables," 2020 International Joint Conference on Neural Networks (IJCNN), Glasgow, UK, 2020, pp. 1-8, doi: 10.1109/IJCNN48605.2020.9207703.
